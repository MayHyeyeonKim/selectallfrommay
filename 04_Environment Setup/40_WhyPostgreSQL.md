# Tools Used in This Course

In this course, will focus on two free and widely-used tools:

## 1. PostgreSQL (Post-Ingres(Interactive Graphics and Retrieval System) Structured Query Language)

- Popular relational database software
- Open-source and cross-platform
- Used globally in production environments
- Supported by major cloud providers (Google, Microsoft, etc.)

## 2. Valentina DBE

- Cross-platform database management UI
- Used to manage PostgreSQL databases
- Free and easy to use
- Helps with writing and executing SQL queries

## Why These Tools?

- Free and open-source
- Lightweight and easy to install
- Proven in real-world applications
- Broadly adopted in industry

## Alternative Database Management Tools

These tools are called **Database Management Tools** or **Database Clients**, and they are software that connects to and manages databases through a GUI. Alternatives to Valentina DBE include:

- **DBeaver**: A free open-source tool that supports various databases like PostgreSQL, MySQL, and SQLite. The community edition is powerful and extensible with plugins.
- **pgAdmin**: A free tool dedicated to PostgreSQL, excellent for query execution, backup/restore, and monitoring.
- **DataGrip**: A paid tool from JetBrains, based on IntelliJ, offering powerful query editing and debugging features for various databases.

For MongoDB (a NoSQL database), **MongoDB Compass** serves a similar role, providing query execution, data visualization, and schema analysis. Similar tools include Studio 3T and NoSQLBooster.
